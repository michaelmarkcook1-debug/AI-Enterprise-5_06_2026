# Sprint 02 Master Prompt — Detailed Build

You are continuing work on the existing AI Enterprise Next.js app after Pack 01.

## Objective

Implement the deeper CIO Decision Intelligence capability layer.

The app should now become useful not only for exploring AI vendors, but for making, defending, and monitoring executive AI decisions.

## Priority Order

1. Demonstrate — Board Defence Framework
2. Monitor — Recommendation Drift and Assumption Monitoring
3. Assess — Three Assessment Tiers
4. Understand — Strategic Sustainability and Risk Intelligence
5. Query — Executive Market Briefing
6. Investor Tools — Scenario Simulator and Investment Lens

## Required Shared Concepts

Create or reuse shared types and helper functions for:

- CIO Confidence Score
- Board Defence Score
- Strategic Sustainability Score
- AI Disruption Risk
- Platform Encroachment Risk
- Dependency Risk
- Strategic Optionality
- Assumption Monitoring
- Recommendation Drift
- Reassessment Trigger

## Use Existing Data Where Possible

Use existing data repositories:

- vendors
- momentum
- pillar scores
- news
- market dashboard
- watchlists
- provenance
- reputation
- pricing

Where no real data exists, implement estimated seed data with clear labelling.

## Acceptance Criteria

- All major pages load.
- New sections are visible.
- Existing components still render.
- Every estimated score is labelled.
- Demonstrate feels board-ready.
- Monitor feels like a continuous validity layer.
- Assess clearly exposes three tiers.
- Build passes or exact failures are reported.
